# Microblog Platform (Twitter-like)


A fully featured mini social media platform including posts, likes, comments,
follows, caching, messaging queue and CI/CD.


## Features
- User authentication (JWT + Refresh token)
- Post / Comment / Like
- Follow system
- Feed algorithm
- Trending hashtags
- Notification service
- Redis caching
- Kafka event-driven architecture
- Docker + CI/CD


## Tech Stack
- Backend: Spring Boot 3, Java 17
- Database: PostgreSQL/MySQL
- Cache: Redis
- Messaging: Kafka
- API Spec: OpenAPI 3.0
- Infra: Docker, Kubernetes (optional)
- CI/CD: GitHub Actions
